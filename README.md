# Minesweeper Game

## Overview
This project is a desktop-based Minesweeper game developed using C#. The game allows users to reveal cells, avoid mines, and complete the board.

## Technologies
- C#
- .NET

## Features
- Interactive grid system
- Mine detection logic
- Game over and restart functionality

## Run
Open the project in Visual Studio and run the application.
